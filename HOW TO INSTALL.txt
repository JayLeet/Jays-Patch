JAY'S PATCH - SERVER INSTALLATION

Thank you for installing Jay's Patch.

Jay's Patch is an unofficial server-side add-on for Sybillian's Blood on the
Clocktower modpack. It is not a standalone Minecraft server pack.

REQUIREMENTS

- Sybillian's Blood on the Clocktower modpack version 1.5.4
  https://modrinth.com/modpack/blood-on-the-clocktower/version/1.5.4
- Minecraft Java Edition 1.21.10

FIRST-TIME INSTALL

1. Install Sybillian's Blood on the Clocktower 1.5.4 on the server first.
2. Start and then stop the server completely.
3. Back up your server's current world and config folders.
4. Extract the ZIP on your computer.
5. Replace the server's existing world folder with the included world folder.
6. Copy the included config folder into the server folder, merging it with the
   existing config folder and replacing files when asked.
7. Open the server's existing server.properties file and update only the keys
   listed under REQUIRED SERVER.PROPERTIES VALUES below.
8. Restart the server and wait until it has fully loaded.
9. Join the server and accept the recommended Jay's Patch resource pack when
   prompted.

REQUIRED SERVER.PROPERTIES VALUES

Update these keys in your existing server.properties. Keep every unrelated
server setting already present in that file.

function-permission-level=3
spawn-protection=0
resource-pack=https://download.mc-packs.net/pack/1441b43c2a7300679f10fa1cf2254ce93e922043.zip
resource-pack-sha1=1441b43c2a7300679f10fa1cf2254ce93e922043
resource-pack-id=3c91f0a2-7ecc-443d-b5b2-4b072fe10e78
require-resource-pack=false
resource-pack-prompt={"text"\:"","extra"\:[{"text"\:"BOTC","color"\:"dark_red","bold"\:true},{"text"\:" | ","color"\:"dark_gray","bold"\:false},{"text"\:"Jay's Patch Resource Pack","color"\:"gold","bold"\:false},{"text"\:"\\nAccept this pack to see Jay's Patch's custom icons.","color"\:"gray","bold"\:false}]}

RESOURCE-PACK FALLBACK

The URL above is the official hosted Jay's Patch resource pack. If it stops
working, upload resourcepack/Jays-Patch-resourcepack.zip to a Minecraft
resource-pack host such as MC-Packs. Replace resource-pack,
resource-pack-sha1, and resource-pack-id with the values from the new host.
